Privacy Policy

Mumu Gmail Filter is a personal internal tool. This policy describes how data is handled.

Data Collected

Gmail message metadata (subject, sender) is accessed solely for the purpose of filtering and labeling emails.

Data Usage

Data is used only to organize emails within the owner's Gmail account. No data is stored externally.

Data Sharing

No data is shared with any third parties.
