# Mumu Gmail Filter

**Mumu Gmail Filter** is a personal internal tool for automatically filtering and labeling promotional emails in Gmail.

## Privacy Policy

This application is for personal internal use only.

- **Data collected**: Gmail message metadata (subject, sender) for filtering purposes
- **Data usage**: Used solely to label and organize emails within the owner's Gmail account
- **Data sharing**: No data is shared with any third parties
- **Data retention**: No data is stored outside of Gmail

For questions, contact the repository owner.
